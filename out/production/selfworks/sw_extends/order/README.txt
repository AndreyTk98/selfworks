
- Типы заказов:
    1) Стандартный (фиксированная цена за кг)
    2) Курьером(цена за кг * процент зависящий от расстояния) только по городу
    3) Срочная почтой - два типа:
        а) Доставка внутри страны(Внутри страны, цена зависит от масштаба доставки: город, регион, страна)
        б) Доставка по миру (процент в зависимости от континента)
